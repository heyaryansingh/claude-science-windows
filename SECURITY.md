# Security Policy

## Supported Versions

This repository tracks Anthropic's public Claude Science setup guidance. Use the latest commit or release.

## Reporting A Vulnerability

Open a GitHub issue for script bugs that do not involve secrets.

Do not include:

- Claude account tokens
- API keys
- SSH keys
- Patient data
- Proprietary lab data
- Private research artifacts

For Claude Science product or account security issues, report through Anthropic's official support and security channels.
